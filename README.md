# MetaCred Monorepo

Packages and apps for MetaCred using yarn v2 workspaces

Stack:

- TypeScript
- NextJS
- ChakraUI
- ESLint
- Prettier
- lint-staged + husky (lint + typecheck on push, format code on commit, etc)
- tsup

### Running locally

1. Install packages: `yarn`
2. Run Creditor app: `yarn creditor dev`
